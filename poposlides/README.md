# poposlides
#####poposlides是一个Jquery轮播插件，大小仅5KB

##[DEMO](http://po-po.github.io/poposlides/)

##使用参数
```javascript
auto:true,             //自动播放
nav:true,              //切换按钮
playspeed:3500,        //自动播放速度
fadespeed:500,         //淡入淡出速度
loop:true,             //循环播放
pagination:true,       //页码显示
pagecenter:true,       //页码居中
trigger:"click",       //页面触发类型click/mouseover
prev:".prev",          //上一页按钮
next:".next"           //下一页按钮
```
